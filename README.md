```
To use

# Clone this repository
git clone https://github.com/Ryudnar/HOS

# Go into the repository
cd HOS

# Install dependencies
npm install

# Run scanner.exe first with administrator previlege
# scanner.exe can found in HOS root folder, or HOS back-end github page
# > https://github.com/nulledge/InformationSecurity2017

# Run the app
npm start
```
